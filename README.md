An opinionated list of website with awesome fonts

``` css
    font-family: SFNSText,Helvetica Neue,sans-serif;
```

``` css
}
body {
    min-height: 100%;
    font-family: 'proxima-nova',helvetica neue,helvetica,arial,sans-serif;
    font-size: 16px;
    line-height: 1.5;
    color: #4d5659;
    text-rendering: optimizeSpeed;
}
```
